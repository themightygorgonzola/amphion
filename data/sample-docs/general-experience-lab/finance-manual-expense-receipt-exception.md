# Finance Manual EX-410: Receipt Exception Workflow for Travel Reimbursement

## Applies to

Employee travel and meal reports submitted through the expense platform.

## Use this procedure when

- Card transactions imported but receipts are missing.
- Manager approval is blocked on attachment validation.
- The traveler used the mobile capture flow.

## Procedure

1. Check the receipt image queue for delayed uploads.
2. Link the missing images to the existing report.
3. Document the exception reason for audit review.
4. Re-submit the report for manager approval.

## Escalate to vendor payment operations if

- The issue involves a supplier invoice instead of employee travel.
- A payment posted twice against the same invoice.
- Bank detail changes or PO remittance mismatches are involved.
